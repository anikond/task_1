Задана рекурсивная функция:
f(n)=5⋅f(n−1)+f(n−2),  f(0)=1, f(1)=3

Требуется найти 40-й нечётный элемент этой последовательности (A[39]).

Для решения используется рекурсивная функция f(n) с мемоизацией (lru_cache) для ускорения вычислений. В основном цикле перебираются значения f(n) и отбираются только нечётные. Эти значения сохраняются в массив A. Цикл завершается, когда в массиве накапливается 40 элементов. Затем выводится значение A[39].

Ответ на задачу: 184153577162052268122747461393215875186211
