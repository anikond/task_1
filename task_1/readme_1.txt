В задаче требуется найти минимальное количество точек, которые покрывают все отрезки на числовой прямой, так чтобы каждый отрезок содержал хотя бы одну из этих точек.
Для решения задачи сначала сортируем отрезки по правому концу. Затем проходим по ним слева направо. Если текущий отрезок не покрыт последней выбранной точкой, добавляем в ответ его правый конец как новую точку.

Ответ  на задачу: 18
